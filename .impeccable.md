# Design Context

- Target audience: Primarily technical installers working on-site, with enough clarity that less technical end users can still understand the interface.
- Use cases: Install the device beside a doorway, verify the sensor is healthy, calibrate it in place, inspect live distances and status, and adjust practical tuning values without opening Home Assistant.
- Brand personality and tone: Professional, technical, elegant.
- Visual direction: Clear, structured, calm, and polished. Support both light and dark themes.
- Anti-reference: Avoid cyberpunk, futuristic neon, or overly flashy visual language.
